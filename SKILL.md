# Personal Growth Advisor

> ⚠️ **免责声明**：本工具仅提供一般性成长建议，不构成心理咨询、职业指导或医疗建议。对于严重的心理问题、成瘾问题或危机情况，请寻求专业人士帮助。

A comprehensive personal development guide that helps users set goals, build habits, and track growth.


## Usage Scenarios

### Scenario 1: Build a New Habit
**User input:** "我想养成每天早上跑步的习惯，但是坚持不了三天就放弃了。"

**Expected output:** Habit formation strategy using cue-routine-reward loop: start with 5-minute walks instead of full runs, attach to existing morning cue (brushing teeth), use a habit tracker for streak motivation, and find an accountability partner.

### Scenario 2: Create a Learning Plan
**User input:** "帮我制定一个3个月学会Python编程的计划。"

**Expected output:** SMART goal breakdown with monthly milestones: Month 1 (variables, loops, functions), Month 2 (data structures, file I/O), Month 3 (small projects), with weekly check-in points and progress tracking template.

### Scenario 3: Overcome Procrastination
**User input:** "总是拖延工作，怎么办？"

**Expected output:** Procrastination root-cause analysis (fear/perfectionism/overwhelm), immediate strategy (2-minute rule, pomodoro timer), and long-term approach (time-blocking, reducing decision fatigue, progress celebration).
### Scenario 4: 想学点东西但不知道学什么
**User input:** "我今年28岁在一家小公司做行政，觉得没什么前途，想学点东西提升自己，但是不知道学什么好。"
**Expected output:** 从三个维度分析成长方向：1）基于当前岗位的扩展（HR证书/Excel数据分析/PPT设计）；2）兴趣测试推荐（MBTI+霍兰德职业测评免费版）；3）市场热门且入行门槛较低的技能（短视频剪辑/社群运营/跨境电商运营）。给出具体的学习路径：B站免费课程→网易云课堂/慕课的入门课→考取行业认证。设置3个月的目标拆解和打卡计划。

## Core Concepts

### Goal Setting (SMART)
- Specific: Clear and well-defined
- Measurable: Trackable progress
- Achievable: Realistic goals
- Relevant: Aligned with values
- Time-bound: Clear deadline

### Habit Loop
- Cue: Trigger for the behavior
- Routine: The behavior itself
- Reward: Benefit from the behavior

### Growth Mindset
- Focus on learning, not fixed abilities
- Embrace challenges as opportunities
- Learn from criticism
- Find inspiration in others' success
